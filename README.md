# MSAL OAuth2.0 Demo

MSAL authentication demo using:
- Angular Frontend
- ASP.NET Backend
- Azure Active Directory
- OAuth2 Authorization Code Flow
- PKCE Authentication
---

# Packages Used

## Angular
@azure/msal-browser
@azure/msal-angular
```
## .NET
Microsoft.Identity.Web
```

# Features
- Microsoft Login
- Azure AD Authentication
- OAuth2 + PKCE Flow
- Access Token Handling
- Protected API Calls
- Angular + ASP.NET Integration
---

# Authentication Flow
Angular App
    ↓
Microsoft Login
    ↓
OAuth2 + PKCE
    ↓
Access Token
    ↓
ASP.NET API
